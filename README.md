# LinkSage  

## Features  
- **Custom Short Links** – Every shortened URL follows a consistent format.  
- **Instant Redirection** – Clicking a short link automatically redirects users to the original URL.  
- **Database Storage** – Stores all URLs for future access and tracking.  
- **Minimal & Fast** – Optimized for quick responses and a lightweight experience.  

## Use Cases  
- Share long URLs in a compact format.  
- Track and manage frequently used links.  
- Improve link aesthetics for social media or messaging.  

LinkSage is built using the **MERN stack (MongoDB, Express, React, Node.js)** for a seamless user experience.
